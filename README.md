# Project Name: DigiMart

A professional, full-stack application built to serve as an integrated platform for community support and resource management. This platform provides a seamless experience for users to manage listings, track analytics, and interact with a community.

## 🚀 Technology Stack

* **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS, Recharts.
* **Backend**: Node.js, Express.js, TypeScript.
* **Database**: MongoDB.
* **UI Components**: HeroUI, React Icons.
* **Animation**: GSAP, Lenis Smooth Scroll (Not Placed Yet).

## 🔑 Key Features

* **Authentication**: Secure login/registration with demo credentials and role-based access control.
* **Dashboard**: Real-time analytics tracking with interactive charts and inventory management.
* **Listings Management**: Full CRUD capabilities—add, view, and manage items with skeleton loaders for optimized UX.
* **Explore & Search**: Advanced filtering (category, price, etc.) and sorting with responsive grid layouts.
* **Responsive Design**: Consistent, mobile-first UI with a focus on deep forest green (#244D3F) branding and clean typography.

## 📋 Getting Started

### Prerequisites
* Node.js (v18+)
* MongoDB Cluster connection string
* Environment variables (.env.local):
    * MONGODB_URI
    * JWT_SECRET
    * NEXT_PUBLIC_API_URL

### Installation
1. Clone the repository:
   git clone [https://github.com/arRahat129/digi-mart]
   cd digi-mart
2. Install dependencies:
   npm install
3. Run the development server:
   npm run dev
